# Utils 目录文档

`backend/utils/` 目录包含项目通用的工具类和辅助函数。

## 文件列表

### 1. `background_task_manager.py`
-   **用途**: 管理异步后台任务（Fire-and-Forget 模式）。
-   **功能**:
    -   `add_task`: 提交一个协程作为后台任务运行，而不阻塞当前请求。
    -   **异常处理**: 自动捕获并记录任务中的异常，防止因未捕获异常导致 loop 崩溃。
    -   **优雅关闭**: 提供 `wait_all_pending` 方法，在服务关闭时等待所有后台任务完成（带超时）。
    -   **场景**: 用于日志记录、记忆压缩、数据上报等非关键路径操作。

# 🎮 VillagerAgent 功能清单

> 本文档详细列出 VillagerAgent 能够在 Minecraft 中操控 Bot 执行的所有功能

---

## 📋 功能总览

VillagerAgent 通过 Mineflayer 与 Minecraft 服务器交互，Bot 能够执行 **50+ 种高级动作**，涵盖以下类别：

| 类别 | 功能数量 | 说明 |
|------|---------|------|
| 🚶 移动导航 | 6 | 移动到坐标、玩家、建筑、动物等 |
| ⛏️ 方块操作 | 6 | 挖掘、放置、激活方块等 |
| 📦 物品管理 | 10 | 装备、丢弃、存取、合成物品等 |
| 🔥 熔炼烹饪 | 2 | 熔炉操作、烹饪食物 |
| 👁️ 感知探索 | 5 | 扫描周围、获取环境/实体信息 |
| 🐄 实体交互 | 8 | 攻击、骑乘、驯服、交易等 |
| 💬 社交对话 | 3 | 与玩家/NPC对话、等待反馈 |
| 🏗️ 建筑辅助 | 4 | 泥土梯子、泥土横梁等高空建筑辅助 |
| 📖 阅读书写 | 4 | 阅读告示牌/书本、书写内容 |
| 🎣 特殊技能 | 4 | 钓鱼、睡觉、吃喝等 |
| 🎛️ 容器操作 | 4 | 打开/关闭箱子、获取容器内容 |

## 🚶 移动导航功能

### 1. `navigateTo` - 移动到指定坐标
```python
navigateTo(player_name, x, y, z, emotion, murmur)
```
- **功能**: 移动 Bot 到指定的 x, y, z 坐标位置
- **返回**: `{'message': msg, 'status': True/False}`
- **示例**: 让 Bot 移动到坐标 (100, 64, 200)

### 2. `navigateToPlayer` - 移动到指定玩家
```python
navigateToPlayer(player_name, target_name, emotion, murmur)
```
- **功能**: 移动 Bot 到指定玩家附近
- **用途**: 多智能体协作时跟随或会合

### 3. `navigateToBuilding` - 移动到建筑
```python
navigateToBuilding(player_name, building_name, emotion, murmur)
```
- **功能**: 移动 Bot 到指定名称的建筑位置
- **用途**: 配合建筑标记系统使用

### 4. `navigateToAnimal` - 移动到动物
```python
navigateToAnimal(player_name, animal_name, emotion, murmur)
```
- **功能**: 移动 Bot 到指定动物附近
- **用途**: 农场任务、驯服动物

### 5. `performMovement` - 执行移动动作
```python
performMovement(player_name, action_name, seconds, emotion, murmur)
```
- **功能**: 执行基础移动动作
- **支持动作**: `jump`, `forward`, `back`, `left`, `right`
- **参数**: `seconds` 指定动作持续时间

### 6. `lookAt` - 看向目标
```python
lookAt(player_name, name, emotion, murmur)
```
- **功能**: 让 Bot 看向指定的玩家或物体

## ⛏️ 方块操作功能

### 1. `MineBlock` - 挖掘方块
```python
MineBlock(player_name, x, y, z, emotion, murmur)
```
- **功能**: 在指定坐标挖掘方块
- **自动**: 自动选择合适的工具

### 2. `placeBlock` - 放置方块
```python
placeBlock(player_name, item_name, x, y, z, facing, emotion, murmur)
```
- **功能**: 在指定位置放置方块
- **facing参数**: `W`, `E`, `S`, `N`, `x`, `y`, `z`, `A`(自动)
- **返回**: `{'message': msg, 'status': True/False}`

### 3. `useItemOnBlock` - 在方块上使用物品
```python
useItemOnBlock(player_name, item_name, x, y, z, emotion, murmur)
```
- **功能**: 在指定方块上使用物品
- **示例用途**:
  - 锄头在泥土上 → 变成耕地
  - 种子在耕地上 → 种植作物
  - 水桶在方块上 → 放水
  - 矿车在铁轨上 → 放置矿车

### 4. `ToggleAction` - 激活/切换方块
```python
ToggleAction(player_name, item_name, x, y, z, emotion, murmur)
```
- **功能**: 开关门、拉杆、按钮等可交互方块
- **支持**: 栅栏门、木门、拉杆、按钮
- **不支持**: 压力板(需站在上面)、铁门(需红石信号)

### 5. `erectDirtLadder` - 搭建泥土梯子
```python
erectDirtLadder(player_name, top_x, top_y, top_z, emotion, murmur)
```
- **功能**: 从地面向上搭建泥土柱子到指定高度
- **用途**: 高空建筑时的脚手架

### 6. `dismantleDirtLadder` - 拆除泥土梯子
```python
dismantleDirtLadder(player_name, top_x, top_y, top_z, emotion, murmur)
```
- **功能**: 从顶部向下拆除泥土柱子

## 📦 物品管理功能

### 1. `equipItem` - 装备物品
```python
equipItem(player_name, slot, item_name, emotion, murmur)
```
- **功能**: 将物品装备到指定槽位
- **槽位**: `hand`, `head`, `torso`, `legs`, `feet`

### 2. `tossItem` - 丢弃物品
```python
tossItem(player_name, item_name, count, emotion, murmur)
```
- **功能**: 丢弃指定数量的物品到地面

### 3. `handoverBlock` - 递交物品
```python
handoverBlock(player_name, target_player_name, item_name, item_count, emotion, murmur)
```
- **功能**: 将物品递交给其他玩家
- **自动**: 自动检查物品数量，自动移动到目标玩家

### 4. `withdrawItem` - 从容器取出物品
```python
withdrawItem(player_name, item_name, from_name, item_count, emotion, murmur)
```
- **功能**: 从箱子/容器/熔炉中取出物品
- **from_name**: `chest`, `container`, `furnace`

### 5. `storeItem` - 存入物品到容器
```python
storeItem(player_name, item_name, to_name, item_count, emotion, murmur)
```
- **功能**: 将物品存入箱子/容器

### 6. `craftBlock` - 合成物品
```python
craftBlock(player_name, item_name, count, emotion, murmur)
```
- **功能**: 在工作台上合成物品
- **自动**: 自动查找合成配方

### 7. `enchantItem` - 附魔物品
```python
enchantItem(player_name, item_name, count, emotion, murmur)
```
- **功能**: 在附魔台上附魔物品

### 8. `repairItem` - 修复物品
```python
repairItem(player_name, item_name, material, emotion, murmur)
```
- **功能**: 在铁砧上修复物品

### 9. `wear` - 穿戴装备
```python
wear(player_name, slot, item_name, emotion, murmur)
```
- **功能**: 穿戴指定装备到指定槽位

### 10. `fetchContainerContents` - 查看容器内容
```python
fetchContainerContents(player_name, item_name, position, emotion, murmur)
```
- **功能**: 获取容器中的物品列表
- **item_name**: `chest`, `inventory`, `furnace`, `container`
- **返回**: `{'data': [{'name': name, 'count': count}, ...], 'status': True/False}`

## 🔥 熔炼烹饪功能

### 1. `SmeltingCooking` - 熔炼/烹饪
```python
SmeltingCooking(player_name, item_name, item_count, fuel_item_name, emotion, murmur)
```
- **功能**: 在熔炉中熔炼或烹饪物品
- **参数**: 
  - `item_name`: 要熔炼的物品
  - `item_count`: 数量
  - `fuel_item_name`: 燃料(如 `coal`, `planks`)

### 2. 相关的容器操作
- 熔炉支持通过 `withdrawItem` 和 `storeItem` 操作

## 👁️ 感知探索功能

### 1. `scanNearbyEntities` - 扫描周围实体
```python
scanNearbyEntities(player_name, item_name, radius, item_num, emotion, murmur)
```
- **功能**: 在指定范围内搜索方块、物品、生物、箱子等
- **返回**: `{'data': [{'x': x, 'y': y, 'z': z}, ...], 'status': True/False}`
- **注意**: 无法搜索箱子/容器/背包内的物品

### 2. `get_environment_info` - 获取环境信息
```python
get_environment_info(player_name, emotion, murmur)
```
- **功能**: 获取当前环境信息
- **返回**: 时间、天气等信息

### 3. `get_entity_info` - 获取实体信息
```python
get_entity_info(player_name, target_name, emotion, murmur)
```
- **功能**: 获取指定实体的详细信息
- **返回**: 实体名称、位置(x,y,z)、手持物品等

### 4. `read` - 阅读告示牌/书
```python
read(player_name, item_name, emotion, murmur)
```
- **功能**: 阅读附近的告示牌或书本内容

### 5. `readPage` - 阅读书页
```python
readPage(player_name, item_name, page, emotion, murmur)
```
- **功能**: 阅读书本的指定页

## 🐄 实体交互功能

### 1. `attackTarget` - 攻击目标
```python
attackTarget(player_name, target_name, emotion, murmur)
```
- **功能**: 攻击最近的指定名称实体
- **用途**: 战斗、获取掉落物

### 2. `useItemOnEntity` - 对实体使用物品
```python
useItemOnEntity(player_name, item_name, entity_name, emotion, murmur)
```
- **功能**: 对实体使用指定物品
- **示例**:
  - 骨头 + 狼 → 驯服狼
  - 水桶 + 牛 → 收集牛奶
  - 剪刀 + 羊 → 剪羊毛
  - 马鞍 + 马 → 给马装鞍
  - 胡萝卜钓竿 + 猪 → 控制猪

### 3. `mountEntity` - 骑上实体
```python
mountEntity(player_name, entity_name, emotion, murmur)
```
- **功能**: 骑上马、猪、船、矿车等

### 4. `dismountEntity` - 下马
```python
dismountEntity(player_name, emotion, murmur)
```
- **功能**: 从当前骑乘的实体下来

### 5. `rideEntity` - 乘坐实体
```python
rideEntity(player_name, entity_name, emotion, murmur)
```
- **功能**: 乘坐实体(与mount类似)

### 6. `disrideEntity` - 停止乘坐
```python
disrideEntity(player_name, emotion, murmur)
```
- **功能**: 停止乘坐实体

### 7. `trade` - 交易
```python
trade(player_name, item_name, with_name, count, emotion, murmur)
```
- **功能**: 与村民NPC进行交易
- **返回**: 交易物品和数量的详情

### 8. `eat` - 进食
```python
eat(player_name, item_name, emotion, murmur)
```
- **功能**: 吃食物恢复饥饿值

## 💬 社交对话功能

### 1. `talkTo` - 对话
```python
talkTo(player_name, entity_name, message, emotion)
```
- **功能**: 向指定玩家发送消息
- **特性**: 支持表情符号
- **注意**: 必须指定具体的玩家名称

### 2. `waitForFeedback` - 等待反馈
```python
waitForFeedback(player_name, entity_name, seconds, emotion, murmur)
```
- **功能**: 等待其他玩家的回复
- **参数**: `seconds` 等待时间(默认10秒)

### 3. `chat` - 聊天广播
```python
agent.chat(msg, async_tag=False)
```
- **功能**: 在游戏中发送聊天消息
- **async_tag**: 是否异步发送

## 🎯 任务类型支持

VillagerAgent 通过以上功能组合，支持以下复杂任务类型：

### 1. 🏗️ 建筑任务 (Construction)
- 按蓝图建造建筑
- 多智能体协作分工
- 脚手架自动搭建/拆除

### 2. 🌾 农业任务 (Farming)
- 开垦土地、播种
- 作物收获
- 动物养殖和繁殖

### 3. 🔐 逃脱室谜题 (Puzzle)
- 阅读提示
- 操作机关
- 寻找物品

### 4. ⚔️ 战斗任务
- 自动攻击目标
- 装备武器盔甲

### 5. 🤝 协作任务
- 物品传递
- 任务分工
- 实时对话协调

## ⚙️ 参数说明

所有功能共享以下通用参数：

| 参数 | 类型 | 说明 |
|------|------|------|
| `player_name` | str | Bot 的名称 |
| `emotion` | list | 表情符号列表，如 `['😊', '👍']` |
| `murmur` | str | Bot 的内心独白/自言自语 |

### 返回值格式

大多数功能返回类似格式：
```json
{
    "message": "操作结果描述",
    "status": true/false,
    "data": [...] // 可选的数据返回
}
```
